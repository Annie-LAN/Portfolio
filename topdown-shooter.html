Top-down Shooter

Developed a top-down perspective shooting game using C# and Unity Engine where the player shoots waves of chasing enemies using various types of guns.

Don't let the chasing enemies get you. Don't fall from the map. Survive to the last. See what's the highest score you can get!

Controls

  • WASD to move
  • Left click to shoot

Download and Install

Supported on Windows, Mac, and Linux.

  • Download

Install Instructions:

  1. Unzip the file
  2. Open the respective file for your platform:
    • Windows: "Top-down Shooter v1.exe"
    • Mac: "Top-down Shooter v1.app"
    • Linux: "Top-down Shooter v1.x86_64"
